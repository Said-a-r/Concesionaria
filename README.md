# Concesionaria 
Aplicación web para la gestión de inventario de una concesionaria de autos. Permite realizar operaciones CRUD (Crear, Leer, Actualizar, Eliminar) sobre vehiculos, con subida de imágenes, filtros de búsqueda y una interfaz moderna con Bootstrap.

## Requisitos previos
| **Python** | 3.10 o superior 
| **MongoDB Atlas** | Cuenta gratuita
